# yoo.ie

Website for Yoo.ie 

## Run dev build & start dev server

`yarn dev`

## Run prod build in dev environment

`yarn dev-build`

## Run prod build

`yarn build`

## Start prod-like server with prod build

`yarn start`

## Format code

`yarn format`

## Lint code

`yarn lint`

## Type-check code

`yarn type-check`

## Test code

`yarn test`

## Test watcher

`yarn test:watch`

## Lint, Type-check & Test code

`yarn test-all`
